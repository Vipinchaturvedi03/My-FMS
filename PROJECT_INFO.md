# FMS - Farm Management System

**Developer:** Vipin Chaturvedi

## Project Overview

FMS (Farm Management System) ek comprehensive farm management application hai jo kisanon ko unke daily farm operations manage karne mein help karta hai.

## Features

- **Expense Management** - Kharcha track karein (Seed, Cultivation, Fertilizer, Transport)
- **Labor Management** - Mazdoor wages aur pending payments
- **Stock Management** - Inventory with low stock alerts
- **Crop Management** - Fasal planning, calendar, monitoring
- **Reports** - CSV/PDF export
- **Bilingual** - Hindi aur English support

## Tech Stack

- **Backend:** Node.js, Express, PostgreSQL
- **Frontend:** React, Vite, Tailwind CSS
- **Auth:** JWT

## Setup

1. Backend: `cd backend && npm install && npm run migrate && npm start`
2. Frontend: `cd frontend && npm install && npm start`

## Database

PostgreSQL required. Schema: `backend/schema.sql`
